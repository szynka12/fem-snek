Finite elements mesh¶

class femsnek.mesh.feMesh.FeMesh(info: str, nodes: numpy.ndarray, element_lists: list, physical_ids: list)¶

Class containing information about finite element mesh.

This class contains information about finite element mesh. Attributes:

  • _info: string - contains information about the mesh, input format etc
  • _nodes: np_array - table with node coordinates, its shape is equal to (3,nNodes)
  • _internalMesh: touple<femsnek.feMesh.mesh> - internal mesh objects
  • _boundary mesh: touple<femsnek.feMesh.mesh> - boundary mesh objects

Each mesh partition can be described by region double: (‘i’/’b’, N): - ‘i’ means internal and ‘b’ boundary - N number of the mesh
